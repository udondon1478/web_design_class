第7回 配色

それぞれの言葉のイメージから、配色(色のパターン)を作りましょう。

3,4色で配色をイメージし、色をCSS「07-kadai.css」で付けてください。サンプルを用意していますのでそれを参考にしてください。

サンプル

グレースケール(.sample)

CSSは以下のように書いています。


.sample div:nth-child(1) {
  /* base */
  width: 70%;
  background: #eee;
}
.sample div:nth-child(2) {
  /* main */
  width: 25%;
  background: #ccc;
}
.sample div:nth-child(3) {
  /* accent */
  width: 5%;
  background: #999;
}

課題1 - 四季をイメージする配色

春 (.haru)

夏 (.natsu)

秋 (.aki)

冬 (.fuyu)

ポップで軽い (.karui)
